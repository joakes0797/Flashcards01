What is Git?

Git is a tool to manage your source code history. It is a Distributed Version Control System.

What is GitHub?

GitHub is a hosting service for your code. It is a remote repository.

What Git command adds a single file to the staging area?

git add <fileName>

What Git command adds all modified files to the staging area?

git add .

What is the difference between git clone and git pull?

git clone is for putting a copy of a remote repository on your machine for the FIRST time.
git pull is used AFTER cloning a copy, and it pulls from remote to local so you have the most up-to-date version of COLLABORATIVE code.

What is the difference between git push -u origin main and git push?

git push -u origin main sends committed changes to remote repository for the FIRST time.
git push is used in any subsequent sending to remote.

What is the difference between git branch <featureName> and git checkout -b <featureName>?

git branch <featureName> creates a new branch named featureName that is separate from the original code.
git checkout -b <featureName> creates a new branch named featureName AND switches to that branch at the same time.

What Git command compares changes in the working directory against a previously committed version?

git diff

What is the difference between git status and git branch?

git status displays the state of the working directory and the staging area.
git branch lists the branches in the repository.

How do you delete a branch?

git branch -d <branchName>
